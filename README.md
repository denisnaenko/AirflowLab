# AirflowLab
Airflow Lab

1. Создайте и активируйте виртуальное окружение:
```
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или 
venv\Scripts\activate     # Windows
```

2. Установите зависимости:
```
pip install -r requirements.txt
```

3. Запустите Airflow:
```
airflow standalone
```
